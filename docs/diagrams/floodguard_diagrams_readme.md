# FloodGuard simplified poster diagrams

These Mermaid diagrams are simplified for narrow poster columns.

## Recommended use
- `floodguard_architecture_simple.mmd`  
  Use as the main technical figure.

- `floodguard_trust_logic_simple.mmd`  
  Use as a smaller support figure titled **Trust layer**.

- `floodguard_risk_engine_simple.mmd`  
  Use as a smaller support figure titled **Explainable risk logic**.

## Poster advice
For a tight 3-column poster:
1. Put the architecture diagram in the centre column.
2. Put either the trust diagram or risk diagram below it.
3. If space is very limited, keep only architecture + risk.
